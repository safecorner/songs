# 呜呼
词：safecorner		曲：suno.com
```
某某书记，芝麻小吏
真理在手，无尽权力
涉嫌的罪，抓进监狱
调查文章，顺逆的戏

可能字句像火，点亮黑夜的路
或许真相像光，刺破沉默的雾
可忽然之间，画面开始模糊
像常坏的摄像头，像执法的围挡布

谁在拉下帷幕，把阳光遮住
谁在高墙之后， 把呐喊的嘴紧捂
清理人和物，又是岁月静好
封锁住消息，演升平的歌舞

异议言论，皆是犯上忤逆
风声一动，便是雷霆震怒
公检法的章，层层加固
黑白之间，界线变得含糊

诬告陷害，罪名写得仓促
非法经营，帽子扣得突兀
异地清晨，警笛声急促
记者的背影，消失在晨雾

若是清白在手，何必惊弓之弩
若是证据在握，何须暗度陈仓路
堂堂正正，在法庭上陈述
让法律裁决，而非封言路

真相不该被剪成静音的图
文字不该被锁进无名的库
当正义被推向无人的角落
依旧还有人为众生鼓与呼

镜头也许会坏，画面也许会糊
可记忆不会删除，风声仍在传出
总有人守着那一束光不服输
等天亮的时候，让世界看清楚

呜呼 ......
呜呼 ......
```
