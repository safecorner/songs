# 生于此地

 [Intro]
    [Fingerstyle acoustic guitar]
    [Slow, lonely]

    镜子里那张脸
    突然让我想起你们
    那么穷的年岁里
    为什么要生下我？
    
    十分钟的欢愉
    换我一生的跋涉
    没给我吃饱的饭
    却给了出人头地的枷锁

    [Bass enters, heavy, low]
    我听了
    我努力了
    没跟别人比吃喝
    没学会自私
    千军万马里挤过独木桥
    只换来一张纸
    你们又说
    要会来事
    要当官
    
    [Chorus]
    [Light percussion, restrained emotion]
    可我没有门
    也没有窗
    出身农村的我
    连心仪的姑娘都不敢望
    我看着不如我的人
    步步高升
    我连个像样的婚姻
    都凑不齐

    他们说要走正道
    不要瞎折腾
    可正道走到底
    不过是重复父辈的蹉跎
    
    我想走自己的路
    就成了逆子
    亲戚朋友谁都看不起
    说我不孝
    说我荒唐

    后来我终于走了

    而我的父母
    只能靠我
    
    [Bridge]
    [Instruments drop out]

    我没有做不孝子
    我给了你们超出亲戚的体面

    花了孝子一辈子都赚不来的钱
    把你们送走那天
    我想为自己活一次
    却发现自己也老了
    没机会了

    如果
    你们没死死拖住我
    没给我戴上道德的枷锁

    [Fade instruments]
    [Clean vocal, guitar harmonics]

    最后还是因为瞻前顾后

    去了很多地方
    见了很多人
    做了很多事
    才明白

    人生的悲剧

    而是
    你不应该
    生于此地
